2. Página Web

Una Página Web es conocida como un documento de tipo electrónico, el cual contiene información digital, con datos visuales y/o sonoros, o una mezcla de ambos, a través de textos, imágenes, gráficos, audio o vídeos y otros tantos materiales dinámicos o estáticos. Toda esta información se ha configurado para adaptarse a la red informática mundial, también conocida como World Wide Web.

Las páginas web o como también son conocidas las web page por su nombre en inglés se encuentran contenidas dentro de los sitios web o websites, que son mejor conocidos por los desarrolladores con un nombre de dominio, que almacenan o alojan el contenido que se desarrollan para ser visualizados o utilizados por millones de usuarios.
