DESCRIPCIÓN DEL CURSO

El principal propósito de este curso es introducir al estudiante al diseño y creación Web de uso diario, con especial enfoque en el uso de los estandares de la W3C.

En un primer acercamiento, se estudiarán los conceptos básicos del ambiente Web, las tipologías de sitios web y los criterios de calidad más importantes a considerar para el diseño, desarrollo y evaluación de un sitio Web.    Posteriormente, se dedicarán a aplicar los criterios de calidad para desarrollar sitios Web, que cumplan con los estándares W3C.

Finalmente,  se considerarán los procesos necesarios para alojar, redireccionar y promover un sitio Web en Internet.

La asignatura ha sido dividida en tres (3) módulos:

MÓDULO CONTENIDO SEMANAS
1 Criterios de Evaluación de Sitios Web  2
2 Diseño y Publicación de Sitios Web 8
3 Desarrollo de Sitios Web con HTML 5 Y CSS 4

La asignatura Diseño y desarrollo de páginas Web se dicta de manera presencial. La plataforma e-ducativa es un mecanismo de apoyo a la labor docente y no sustituye hasta este momento las clases presenciales.
