M1 - Terminología Web

Glosario

Módulo 1 - Sitios Web

  • Sitio web: Conjunto de páginas web relacionadas entre sí y accesibles a través de una dirección web única.
  • Diseño web: Proceso de planificación, diseño y creación de sitios web.
  • Desarrollo web: Proceso de creación de sitios web, que incluye la programación y la implementación de funcionalidades.
  • Anatomía de un sitio web: Estructura y componentes de un sitio web, como la cabecera, el menú de navegación, el contenido y el pie de página.
  • Organización de un sitio web: Forma en que se estructura y se presenta la información en un sitio web.
  • Navegación de un sitio web: Forma en que los usuarios se desplazan por un sitio web para acceder a la información que necesitan.
  • Tipologías de sitios web: Clasificación de los sitios web según su finalidad, como los sitios web corporativos, los sitios web de comercio electrónico y los sitios web de entretenimiento.
  • Metodologías de desarrollo web: Conjunto de técnicas y herramientas utilizadas para desarrollar sitios web de manera eficiente y efectiva.
  • Criterios de calidad para el diseño y desarrollo de sitios web: Aspectos a considerar para asegurar que un sitio web cumpla con los estándares de calidad y sea fácil de usar.
  • Alojamiento web: Proceso de almacenamiento y publicación de un sitio web en un servidor para que sea accesible a través de Internet.
  • Promoción web: Estrategias y técnicas utilizadas para aumentar la visibilidad y el tráfico de un sitio web en Internet.

Módulo 2 - Lenguajes de marcado

  • Lenguaje de marcado de hipertexto: Lenguaje utilizado para crear páginas web, que permite definir la estructura y el contenido de una página web.
  • HTML (Lenguaje de marcado de hipertexto): Lenguaje de marcado de hipertexto utilizado para crear páginas web.
  • XHTML (Lenguaje de marcado de hipertexto extensible): Versión más estricta y extensible de HTML.
  • CSS (Hojas de estilo en cascada): Lenguaje utilizado para definir el estilo y la presentación de una página web.
  • Sintaxis: Conjunto de reglas y convenciones que definen la estructura y el formato de un lenguaje de programación.
  • Elementos: Componentes básicos de un lenguaje de marcado de hipertexto, como las etiquetas y los atributos.
  • Etiquetas: Elementos de un lenguaje de marcado de hipertexto que se utilizan para definir la estructura y el contenido de una página web.
  • Atributos: Propiedades de una etiqueta que se utilizan para definir su comportamiento y apariencia.
  • Documentos bien formados: Documentos que cumplen con las reglas y estándares de un lenguaje de marcado de hipertexto.
  • Validación de código: Proceso de comprobar que un documento cumple con las reglas y estándares de un lenguaje de marcado de hipertexto.

Módulo 3 - Hojas de Estilo CSS

  • Hojas de estilo CSS: Lenguaje utilizado para definir el estilo y la presentación de una página web.
  • Diseño web adaptable: Diseño de sitios web que se adapta a diferentes tamaños de pantalla y dispositivos.
  • Diseño web responsivo: Diseño de sitios web que se adapta a diferentes tamaños de pantalla y dispositivos, y que también tiene en cuenta las capacidades del dispositivo.
  • Diseño web líquido: Diseño de sitios web que se adapta a diferentes tamaños de pantalla y dispositivos, y que utiliza porcentajes en lugar de medidas fijas.
  • Diseño web flexible: Diseño de sitios web que se adapta a diferentes tamaños de pantalla y dispositivos, y que utiliza unidades de medida relativas.
  • Diseño web escalable: Diseño de sitios web que se adapta a diferentes tamaños de pantalla y dispositivos, y que utiliza vectores en lugar de imágenes fijas.
  • Diseño web plano: Diseño de sitios web que utiliza elementos planos y minimalistas.
  • Diseño web con capas: Diseño de sitios web que utiliza capas superpuestas para crear efectos visuales.
  • Diseño web con animaciones: Diseño de sitios web que utiliza animaciones para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con tipografía: Diseño de sitios web que utiliza la tipografía para crear efectos visuales y mejorar la legibilidad.
  • Diseño web con imágenes: Diseño de sitios web que utiliza imágenes para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con iconos: Diseño de sitios web que utiliza iconos para crear efectos visuales y mejorar la navegación.
  • Diseño web con colores: Diseño de sitios web que utiliza colores para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con patrones: Diseño de sitios web que utiliza patrones para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con gradientes: Diseño de sitios web que utiliza gradientes para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con sombras: Diseño de sitios web que utiliza sombras para crear efectos visuales y mejorar la legibilidad.
  • Diseño web con bordes: Diseño de sitios web que utiliza bordes para crear efectos visuales y mejorar la legibilidad.
  • Diseño web con efectos de desplazamiento: Diseño de sitios web que utiliza efectos de desplazamiento para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con efectos de transición: Diseño de sitios web que utiliza efectos de transición para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con efectos de animación: Diseño de sitios web que utiliza efectos de animación para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con efectos de paralaje: Diseño de sitios web que utiliza efectos de paralaje para crear efectos visuales y mejorar la experiencia del usuario.
  • Diseño web con efectos de desenfoque: Se refiere a la técnica utilizada para crear un aspecto borroso o difuminado en elementos visuales, como imágenes, fondos o contenedores, con el fin de agregar profundidad, enfocar la atención en ciertas áreas o crear un estilo visual atractivo.
