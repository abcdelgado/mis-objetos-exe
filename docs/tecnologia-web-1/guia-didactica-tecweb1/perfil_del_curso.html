PERFIL DEL CURSO

La Web es un medio de gran importancia tanto en el ámbito educativo, por su capacidad interactiva y multimedia, como en el empresarial por la posibilidad de estar en contacto y comunicación con los clientes de forma permanente.
El conocimiento de tecnologías Web constituyen una herramienta poderosa para el desenvolvimiento profesional del futuro Licenciado en Informática para la Gestión Educativa y Empresarial, capacitándolo para diseñar y desarrollar Sitios Web que cumplan con los estándares, así como para evaluarlas atendiendo a criterios de calidad; y le servirá de base para el desarrollo de aplicaciones web dinámicas y para el comercio electrónico que se tratan en asignaturas posteriores, permitiéndoles el logro de competencias para desarrollar sistemas de información basados en tecnología Web que atiendan a los estándares de accesibilidad y usabilidad.

Al finalizar el curso, el estudiante debe estar en capacidad de:

  • Identificar, combinar y gestionar adecuadamente los estándares para el diseño y desarrollo de sitios Web, así como de validar su uso adecuado.
  • Utilizar herramientas de edición para el diseño y desarrollo de Sitios Web.
  • Reconocer los distintos elementos que participan el el diseño, desarrollo y publicación de Sitios Web.
